== Description ==

Are you selling course from your website. This plugin is best and yes very small but full feature for selling coures.

This provide a custom post type courses and a clean design for the font end.

When we add course we also provide one link where user can purcahse the course and complete.

It ask following information from the admin and publish to the front page


 Short Title

== Installation ==

1. Download and extract   to `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. "Dashboard"->"Settings"->"Saksh Course plugin"
4. After activation one course content type will start showing.
